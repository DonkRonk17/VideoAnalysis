## Feature Icons

### Frame Extraction
```
Simple icon showing video frames being extracted.
A film strip with frames floating out, 
connected by dotted lines.
Cyan accent on dark blue background.
Minimal, clean, 256x256.
```

### Delta Change Detection
```
Icon showing movement/change detection between frames.
Two overlapping rectangles with highlighted differences,
delta symbol or wave form showing change.
Electric blue glow effect.
Minimal, clean, 256x256.
```

### Scene Detection
```
Icon representing scene detection.
A timeline with marked scene boundaries,
scissors or cut marks at transitions.
Gradient from dark to cyan.
Minimal, clean, 256x256.
```

### OCR Text Extraction
```
Icon for text extraction from video.
A video frame with text being lifted out,
magnifying glass or selection highlight.
Clean lines, tech aesthetic.
Minimal, clean, 256x256.
```
